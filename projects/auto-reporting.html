Auto Reporting Module

Part of FW Tools Ecosystem Quota Report Real-time & Scheduled Email Delivery

Tổng quan giải pháp

Module Auto Reporting tự động kết nối với API của nền tảng khảo sát (như Alchemer) để cập nhật số lượng mẫu hoàn tất (Completes) theo từng phút. Ngoài dashboard real-time, hệ thống còn tự động gửi báo cáo qua email theo lịch trình đã thiết lập sẵn.

Feature: Multi-schedule Dispatching

Hệ thống gửi báo cáo tự động (Scheduled Email)

  • 08:00 AM: Gửi Overview Report cho toàn bộ team liên quan (CS, QC, DP).
  • 12:00 PM: Gửi Quota Warning cho bộ phận điều phối (Fieldwork team) để kịp thời bù mẫu.
  • 04:00 PM: Gửi Báo cáo tổng kết ngày cho Project Manager.

Tính năng chính

Real-time Data Sync

Tự động đồng bộ hóa dữ liệu từ Alchemer API mỗi 5 phút, loại bỏ việc xuất file Excel thủ công.

Pacing Engine

Thuật toán tính toán tốc độ thu thập mẫu dự kiến và so sánh với Timeline dự án để đưa ra cảnh báo sớm.
